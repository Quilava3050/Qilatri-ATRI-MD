## 🛠 Configuration

### Basic Setup

javascript
// config.js
export default {
  // Bot Info
  botname: 'ATRI-MD',
  ownername: 'Your Name',
  
  // API Keys
  openai_apikey: 'your_openai_key',
  gemini_apikey: 'your_gemini_key',
  
  // Features
  autoread: false,
  antispam: true,
  premium: true
}


### Advanced Configuration

<details>
<summary>Database Setup</summary>

javascript
// Database configuration
{
  "users": "database/users.json",
  "groups": "database/groups.json",

}


</details>

## 🔧 Troubleshooting

<details>
<summary>❓ Common Issues</summary>

*Q: Bot not responding?*
- Check internet connection
- Verify WhatsApp session
- Restart bot process

*Q: Audio effects not working?*
- Install FFmpeg: npm install ffmpeg
- Check file permissions
- Verify audio file format

*Q: AI not responding?*
- Check API keys in config
- Verify API quotas
- Test API endpoints

</details>
